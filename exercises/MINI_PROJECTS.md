# Mini Projects

## Build an Analytics Dashboard (SQL only)

Pick a dataset (Ecommerce, rideshare, movies) and write queries for the following

* Daily active users
* Revenue by week
* Retention (users who returned after 7 days)
* Top 10% of users by spend

## Recreate a LeetCode-style SQL Challenge Set

Pick 10-20 problems:

* Find the second highest salary
* Find customers who never ordered
* Find employees earning more than their managers
* Find the most active user per month

## Build a CRUD Application and write all SQL by hand

Tables:
* Users
* Posts
* Comments

Queries:
* Insert
* Update
* Delete
* List with joins
* Pagination (LIMIT/OFFSET)
