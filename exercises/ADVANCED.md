# Advanced Exercises

## Schema Design

Design a Schema for the following use cases:

* Blog website
* Shopping cart feature
* Habit tracking application
* Recipe application

Create the following SQL:

* Insert Statements
* Queries to fetch related data
* Queries to compute stats

## Query Optimization

Find a slow query:

* Add indexes
* Rewrite joins
* Replace subqueries with window functions
* use EXPLAIN to compare plans

## Data Cleaning / Transformation

Create a messey CSV file with the following:

* Inconsistent dates
* Mixes casing
* null values
* duplicate rows

Perform the following:

* Normalize dates
* Deduplicate users
* Extract domains from emails
* Convert text categores to IDS
